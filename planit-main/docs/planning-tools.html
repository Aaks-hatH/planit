Planning Tools

Every planning tool, explained

The PlanIt workspace has twelve collaborative tools for your event planning team. Here is what each one does, how to use it, and when to reach for it.

Available in Standard and Enterprise modes

All planning tools described here are available in both Standard and Enterprise event modes. Table Service mode has an entirely different interface and does not include these tools.

Chat

The Chat tab is the real-time messaging channel for your planning team. Messages are delivered in milliseconds via WebSocket and stored persistently in MongoDB for the lifetime of the event.

Real-time delivery
Messages appear instantly on all connected screens via Socket.IO. No polling, no refresh required.
Typing indicators
When another participant is composing, their username appears in a typing indicator at the bottom of the chat panel.
Persistent history
All messages from the start of the event load when you open chat. New members can scroll back to catch up.
Rate limited
30 messages per minute per session. If you hit the limit, a brief error is shown and it resets automatically.

Permissions

ActionParticipantOrganiser
Send a messageYesYes
Delete own messageYesYes
Delete any messageNoYes
Keep chat for discussion; use Bulletin for critical info

Important announcements get buried in a busy chat channel. Use the Chat tab for conversation and the Bulletin for things every team member must see. The Bulletin sends a toast notification to all connected sessions regardless of which tab they're viewing.

Tasks

The Tasks tab is a shared to-do list for your entire team. Any participant can create, assign, prioritise, and complete tasks. Updates broadcast live to every connected session.

  1. 1
    Create a task

    Click the + or Add Task button. Fill in a title (required), optional description, assignee name (free text — doesn't need to be a workspace participant), due date, and priority level.

  2. 2
    Set priority

    Choose High (red), Medium (amber), or Low (blue). High-priority tasks sort to the top of the list automatically.

  3. 3
    Mark complete

    Click the circle to the left of the task. Completed tasks move to the bottom with strikethrough styling. This updates live for everyone.

  4. 4
    Delete a task

    Click the trash icon. The deletion removes it from the database immediately and it disappears from all connected sessions.

The top of the task panel shows a live summary: total tasks, completed count, pending count, and completion percentage. Overdue tasks (past due date, not yet complete) are highlighted with an amber indicator.

Polls

The Polls tab enables real-time group decision-making. Any participant can create a poll, and vote counts update live on every screen as ballots come in.

Notes

The Notes tab is a shared scratchpad styled as a grid of colour-coded sticky notes. Unlike chat, notes are persistent structured items meant to be revisited rather than scrolled past.

Yellow
Open items, pending decisions
Blue
Reference information
Red
Blockers, urgent issues
Green
Confirmed / resolved items
Purple
Ideas and brainstorming
Orange
Vendor and logistics info

These colour meanings are suggestions, not enforced by PlanIt. Develop your own team convention — the important thing is that everyone on your team uses the same system.

Bulletin (Announcements)

The Bulletin tab is a one-way broadcast channel from organiser to team. Only organisers can post; all participants read. Important announcements trigger a toast notification on every connected session regardless of which tab they're viewing.

Use the Important flag for urgent communications

When posting, check the Important box for venue changes, schedule shifts, or emergency notices. Important announcements render with a red border in the Bulletin and trigger the toast notification system.

Budget & Expenses

The Budget tab is a shared ledger for tracking costs as they're incurred and comparing total spending against a target budget.

  1. 1
    Set a budget (organiser only)

    Click Set Budget and enter the total amount. The panel then shows spent vs budget and a remaining balance that turns red when over.

  2. 2
    Log an expense (anyone)

    Click Add Expense. Enter: title, amount, category, who paid, and optional notes. The entry appears instantly for all connected sessions.

  3. 3
    Review the breakdown

    A category summary derives automatically from the labels you use. No pre-configuration needed — just be consistent with your category names.

Amounts are currency-agnostic

PlanIt stores amounts as plain numbers without enforcing a currency symbol. Treat all entries as whatever currency your event is budgeted in. This is a collaborative ledger, not a payment system.

Files

The Files tab lets your team share documents, images, PDFs, spreadsheets, and other files inside the workspace without using an external service.

All uploaded files are permanently deleted along with the rest of the event data. Download anything you need to keep — invoices, contracts, signed documents — before that deadline.

Agenda

The Agenda tab lets you build a structured run-of-show for the event — individual time slots with titles, times, descriptions, and speaker names. All participants can view the agenda; only organisers can add, edit, or delete items.

Items are displayed in chronological order by start time. Use the Agenda to communicate the event flow to team members, share with speakers, and keep everyone aligned on the schedule.

Countdown Timer

A live countdown clock in the workspace header counts down to the event's exact start time, updated every second. It shows days, hours, minutes, and seconds remaining.

When the current UTC time passes the event start, the countdown transitions automatically to a green "Event is Live!" state on all connected sessions simultaneously. No action required.

The countdown is timezone-aware: it calculates from the absolute UTC timestamp set at event creation, so team members in different timezones all see the same remaining time.

Share & Utilities Panel

The Share tab consolidates tools for distributing the event and accessing adjacent systems.

Copy event link
One-click copy of the workspace URL to your clipboard.
Workspace QR code
QR encoding the workspace URL — for getting team members in quickly by scan. Not the same as guest invite QR codes.
Add to Calendar (.ics)
Downloads a standard iCalendar file for Apple Calendar, Google Calendar, Outlook, and others.
Check-in dashboard link
Direct button to the Enterprise check-in dashboard (Enterprise mode only).

People List

The People tab shows every participant currently active in the workspace. Each entry shows the participant's display name, join time, and online status. This is a live view — participants appear when they join and disappear when they leave.

The organiser can see all participants; the total count updates in the workspace header in real time.
