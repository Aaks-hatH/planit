Seating Maps

Visual seating layouts

Build a drag-and-drop floor plan, assign guests to tables, and show staff exactly where to direct each guest at check-in.

On this page

Enabling the seating map

The seating map is available from the check-in page. As organiser, click the Set Up Seating button in the top toolbar. This opens the seating map editor for the first time. The button only appears for organisers when no map has been created yet.

Once a map exists, the button changes to Seating and is visible to all logged-in users (staff see a read-only version, organisers get the full editor).

Building your layout

The seating map editor is a canvas-based tool. Objects snap to a grid and can be freely positioned and resized.

Available objects

ObjectDescription
Round tableCircular table with a configurable label
Rectangular tableLong banquet or boardroom table
StageFixed stage/platform element
Dance floorOpen area marker
Bar / counterService area
Wall / dividerRoom partition or boundary
Text labelFree-form annotation for zones or areas

Editing objects

  • Add: Select an object type from the toolbar and click on the canvas to place it.
  • Move: Drag any object to reposition it.
  • Resize: Drag the corner or edge handles.
  • Label: Click a table to edit its label (e.g. "Table 1", "VIP A").
  • Delete: Select an object and press Backspace or use the delete button in the properties panel.

Saving

Click Save Map to persist your layout. Changes are not auto-saved while editing. Other logged-in users (staff and organiser on other devices) receive the updated map in real-time after you save.

Assigning guests to tables

With the seating map open in editor mode, the right panel lists all unassigned guests. Drag a guest's name from the panel onto a table object to assign them. Alternatively, open the guest's details in the guest list and use the table assignment dropdown.

Once assigned:

  • The table object shows the guest count
  • The guest's invite page shows their table label
  • The wallet card and print badge show the table assignment
  • The boarding-pass review at check-in shows the table with a "Show on Map" shortcut

Unassigning

To remove a table assignment, drag the guest back to the unassigned list, or click the assignment and select Remove.

Check-in integration

Table assignments surface in two places during check-in:

  • The boarding-pass review screen shows the assigned table prominently with a "Show on Map" button
  • The Admitted success screen shows the table label and a "Show on Map" button that opens the seating map zoomed to that table

This removes the need for staff to memorise seating charts — they can direct each guest to their exact table in one tap.

Staff view

Staff (non-organiser logins) see the seating map in read-only display mode. They can zoom and pan to find tables, and they see which guests are assigned where, but they cannot edit the layout or move guests between tables. Only organisers can modify the map.

Real-time sync

The seating map syncs in real-time across all devices on the same event. When the organiser saves a new layout or reassigns a guest, every connected device receives the update immediately via WebSocket. No page refresh is needed.
