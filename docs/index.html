What is PlanIt?

PlanIt is a proprietary freeware event management platform built for organisers who need a complete, professional-grade toolset for managing in-person events. It is free to use through the official hosted service and requires no subscription, no per-event fee, and no feature paywall.

Organisers create events, manage guests, assign seating, and run check-in from a single dashboard. Guests receive a unique invite link — they open it in any browser, see their QR code, and present it at the door. No app downloads. No accounts for guests. No friction.

Platform features

Smart Guest Invites
Personal QR codes, wallet cards, printable A6 badges, RSVP tracking, and optional security PINs per guest.
QR Check-in
Camera scanning, boarding-pass review screen, anti-fraud protection, manager override, and offline queued mode.
Visual Seating Maps
Drag-and-drop floor plans with round tables, rectangular tables, stages, bars, VIP zones, and auditorium layouts.
Walkie-Talkie
WebRTC push-to-talk voice channel connecting all logged-in staff instantly. No separate app or phone number.
Anti-Fraud Engine
Trust scores 0–100, duplicate fingerprint detection, guest flagging, auto-blocking, and a full audit log.
Live Analytics
Real-time attendance counters, check-in progress, group headcounts, and per-role breakdowns updated via WebSocket.
CSV Bulk Import
Upload a spreadsheet of guests with roles, group sizes, PINs, and notes. Template download available in-app.
Offline Mode
Checks-in queue locally when connectivity drops. Auto-syncs when connection returns. Works at venues with weak signal.
Task & Notes Board
Shared task lists and notes visible to all staff logged into the event space. Real-time collaborative updates.

Who is PlanIt for?

PlanIt is built for event organisers who create and manage in-person events. Organisers hold accounts and control everything. Guests receive a unique invite link — they never sign up or log in. Staff use a simple PIN-based login shared by the organiser.

RoleWhat they doNeeds an account?Access level
Organiser Creates events, manages guest lists, configures check-in, sets up seating, views analytics, manages staff Yes Full access
Staff / Check-in Scans QR codes, admits guests, uses walkie-talkie, views read-only seating map PIN-based login (no account) Check-in only
Guest Views their invite page, presents QR code at entrance, RSVPs, saves wallet card No Invite page only

How PlanIt works

PlanIt is a full-stack web application. The frontend is a React single-page app; the backend is a Node.js/Express API with MongoDB persistence. Real-time features — check-in sync, walkie-talkie signalling, seating map updates, and security alerts — are delivered over Socket.IO WebSockets.

Data retention

7-Day Automatic Deletion

All event data — guests, check-in history, messages, files, seating maps, and the event itself — is automatically and permanently deleted seven days after the event date. A countdown warning banner appears in the organiser dashboard as the deletion window approaches. Export anything you need before the deadline.

This policy keeps the platform lean, protects guest privacy, and ensures no long-term data accumulates. It is not configurable — the seven-day window is fixed for all events.

Licensing

Proprietary Freeware

PlanIt is proprietary software. The source code is published for transparency and audit purposes, not for redistribution or self-hosting. You may not fork, modify, self-host, or redistribute PlanIt. All events must run on the official service at planitapp.onrender.com.

Using PlanIt is completely free. There are no subscriptions, no paid tiers, no per-event charges, and no feature paywalls. The platform is operated as a hosted service — you use it through the web app, not by running your own copy.
